# Contributing

Thanks for your interest in contributing! Please follow these guidelines to keep changes consistent and easy to review.

## Setup

1. Install dependencies:
   - `npm install`
2. Copy environment variables:
   - `cp .env.example .env`
3. Start the dev server:
   - `npm run dev`

## Development Notes

- Use the App Router in `src/app`.
- Keep changes scoped and avoid unrelated refactors.
- Run `npm run lint` before submitting a PR.

## Pull Requests

- Describe the problem and the fix.
- Include screenshots or recordings for UI changes.
- Ensure no secrets are committed.

## Reporting Issues

- Provide clear reproduction steps.
- Include logs or error output when possible.
